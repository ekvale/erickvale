CSV Format for Transactions

Your CSV file should include the following columns:

  • transaction_id (required) - Unique transaction identifier
  • date (required) - Date in YYYY-MM-DD or MM/DD/YYYY format
  • amount (required) - Transaction amount
  • description - Transaction description
  • vendor_name - Vendor name (will be created if doesn't exist)
  • department_name - Department name (will be created if doesn't exist)
  • category_name - Budget category name (will be created if doesn't exist)
  • fiscal_year - Fiscal year number
  • invoice_number - Invoice number
  • contract_number - Contract number
  • location - Location/address
  • status - Transaction status (pending, approved, paid, cancelled)
