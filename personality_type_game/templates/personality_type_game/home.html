Personality Type Game

Learn to identify Chase Hughes' six personality types from dialogue transcripts

How to Play

  1. You'll see a short transcript (4-10 lines) with a scenario label
  2. Identify which personality type is dominant (1-6)
  3. Choose the best response option (A-D) to de-escalate or influence effectively
  4. Optionally identify the key "tell" category (control, visibility, belonging, accuracy, freedom, safety)

Scoring

  • +2 points if you correctly identify the personality type
  • +2 points if you choose the correct response
  • +1 bonus point if you identify the tell category correctly
  • -1 point if you use a hint in training mode

The Six Personality Types

1. Driver

Wants: Options, outcomes, speed, autonomy

Tell: Control

2. Expressive

Wants: Recognition, excitement, visibility

Tell: Visibility

3. Relational

Wants: Connection, harmony, appreciation

Tell: Belonging

4. Analyzer

Wants: Clarity, logic, data, definitions

Tell: Accuracy

5. Free Spirit

Wants: Autonomy, flexibility, choices

Tell: Freedom

6. Guardian

Wants: Safety, protocol, risk coverage

Tell: Safety
